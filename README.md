# 🧠 CogniView – AI-Powered Focus Enhancement for Learning

> Enhancing Learning with AI-Based Focus Monitoring

CogniView is an AI-powered web application that helps students and professionals stay focused during study or work sessions by monitoring facial expressions, eye movements, and attention levels in real time. Using Computer Vision and Machine Learning, the system detects drowsiness and distractions, provides instant feedback, and offers personalized productivity insights. :contentReference[oaicite:1]{index=1}

## 🚀 Features

✅ Real-Time Face Monitoring
- Detects eye blinking rate
- Tracks head position
- Monitors facial expressions

✅ AI-Based Focus Detection
- Classifies user into:
  - Focused
  - Distracted
  - Drowsy

✅ Instant Alerts
- Audio notifications
- Visual alerts
- Flash warning for drowsiness

✅ Personalized Dashboard
- Study statistics
- Focus analytics
- Performance tracking

✅ Smart Recommendations
- Suggests study breaks
- Productivity tips
- Focus improvement strategies

✅ Gamification
- Interactive memory games
- Improves engagement after fatigue detection

## 📌 Problem Statement

Students and professionals often lose concentration due to:

- Long screen time
- Mental fatigue
- Drowsiness
- Lack of engagement
- No real-time monitoring in existing learning platforms

CogniView addresses these issues using Artificial Intelligence and Computer Vision to improve focus and learning efficiency. :contentReference[oaicite:2]{index=2}

---

## 💡 Proposed Solution

CogniView continuously analyzes the user's face using a webcam and AI models.

The system:

- Detects eye movement and blinking
- Measures attention level
- Identifies drowsiness
- Sends instant alerts
- Logs focus history
- Generates productivity insights
- Recommends breaks when necessary
